<h1 align="center">Preserve, Reveal, Expand: Faithful 4D Video Editing with Region-Aware Conditioning</h1>
<p align="center">
  <a href="https://arxiv.org/abs/2605.20961">
    <img alt="arXiv" src="https://img.shields.io/badge/arXiv-2605.20961-red">
  </a>
  <a href="https://ricepastem.github.io/PREX-Open/">
    <img alt="Project Page" src="https://img.shields.io/badge/Project%20Page-PREX-blue">
  </a>
</p>

<p align="center">
    TL; DR: PREX is a region-aware framework for faithful 4D video editing. It addresses the Evidence-Role Mismatch problem in existing 4D-driven video diffusion models by decomposing the target spatiotemporal volume into Preserve, Reveal, and Expand roles.
</p>
